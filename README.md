## üìñ Overview

**NEuRoN** (Neural Evolution using Recursive organic Networks) is a high-concept visualization and simulation engine. It models the theoretical evolution of Artificial General Intelligence (AGI) by integrating concepts from:
* **Fractal Geometry:** Recursive neural growth.
* **Quantum Mechanics:** Superposition and entanglement simulation.
* **Biological Morphogenesis:** Cellular differentiation and metabolic pathways.
* **Hyperdimensional Computing:** High-dimensional vector space projections.

This application is built as an interactive dashboard that allows researchers (and you!) to observe how these distinct architectural paradigms evolve, gain "consciousness," and manage entropy over time.

## üß¨ Code Structure

* `ArchitectureType` & `EvolutionPhase` (Enums): Define the states of existence.
* `NeuralNode` (Dataclass): The fundamental atom of the system, containing position, energy, and quantum state.
* `InfiniteArchitectureGenerator`: The "God class" responsible for procedural generation using math (L-systems, Fractals).
* `MetaCognitiveAI`: The brain managing the state, metrics, and logic.
* `AdvancedVisualizationSystem`: The rendering engine using Plotly.
* `LazyLoadingManager`: Optimizes performance by loading heavy visualizers only when needed.
